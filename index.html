Travel{err}


Technologies: MongoDB, Mongoose, Express, Node.js, JavaScript, Angular, CSS

Travel{err} is an application that allows guest users to search for travel destinations based on select criteria. It includes authentication which allows registered users to create the listings, and edit them as desired.

This app was built as a group project. We used a combination of paired and individual programming to complete this. My contributions included creating the flow chart and mockup of our app, some of the styling, and participating in coding the functionality of the app as well.

Open Ocean


Open Ocean is a shared blogging app. Users can read, and like other's entries. They also have the ability to create and edit their own. The entries can be filtered to show only recent or archived based on their status and user preference.

This a full CRUD app built using React and Ruby on Rails. We created our own Rails API for the backend and connected it to our React frontend.

This app was built as a group project. My contributions included creating the React architecture of our app, and coding out the frontend. My partner coded out the backend and took care of most of the design.
