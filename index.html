My Work

REUSE


REUSE is a web application that allows users to buy and/or sell used shoes and clothing. They can create, update or delete listings. Users also have the opportunity to learn more about textile waste when clicking on the links in the homepage.

This a multi-page app built using JavaScript, Node.js, EJS, Express, Mongoose, MongoDB, HTML and CSS.

Visit the live site REUSE or the Github repo.

Travel{err}


Travel{err} is an application that allows guest users to search for travel destinations based on select criteria. It includes authentication which allows registered users to create the listings, and edit them as desired.

This a full CRUD app built using a MEAN Stack (Mongoose, Express, Angular and Node.js.

This app was built as a group project. We used a combination of paired and individual programming to complete this. My contributions included creating the flow chart and mockup of our app, some of the CSS, and participating in coding the functionality of the app as well.

Visit the live site Travel{err} or the Github repo.

Open Ocean


Open Ocean is a shared blogging app. Users can read, and like other's entries. They also have the ability to create and edit their own.

This a full CRUD app built using React and Ruby on Rails. We created our own Rails API for the backend and connected it to our React frontend.

This app was built as a group project. My contributions included creating the React architecture of our app, and coding out the frontend. My partner coded out the backend and took care of most of the design.

Visit the live site Open Ocean or the Github repo.
